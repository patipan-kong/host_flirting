# Avatar Images

Place host avatar images in this directory with the following naming convention:
- host_1.jpg (for host with id: 1)
- host_2.jpg (for host with id: 2)
- etc.

Recommended image specifications:
- Format: JPG or PNG
- Size: 128x128 pixels minimum
- Aspect Ratio: 1:1 (square)
- File size: < 200KB for optimal loading

The app will automatically use placeholder images if the avatar files are not found.
